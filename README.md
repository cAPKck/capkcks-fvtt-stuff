This [FoundryVTT](https://foundryvtt.com) module is a collection of some personal tweaks and extended functionality I use for my Foundry VTT that are too small or too specific to require their own module. If you like any of the stuff, feel free to use this module, but I don't guarantee compatibility. See the table below (may not be complete).

## Installation

1. On Foundry's "Add-on Modules" tab, click "Install Module"
2. At the bottom, paste `https://github.com/cAPKck/capkcks-fvtt-stuff/releases/latest/download/module.json` into the "Manifest URL" field and click "install".

## Features

| **Description** | Notes |
| --- | --- | 
| dnd5e - Sort Spell list by "usable" | Currently requires a tweak in the dnd5e system |
